Application Name: MySampleApp

Description:
This is a sample command-line tool that processes text files,
counts word frequency, and outputs the top 10 most common words.

Requirements:
- Python 3.8 or later
- Required packages:
    - argparse
    - collections

How to Run:
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py --input yourfile.txt --top 10

Example:
   python app.py --input data/article.txt --top 5

Script Options:
--input <filename>      The text file to be processed.
--top <number>          Number of top words to display (default is 10).
--help                  Show usage instructions.

Notes:
- Make sure the input file is UTF-8 encoded.
- This tool ignores case and punctuation.
